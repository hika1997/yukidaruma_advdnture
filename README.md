# ゆきだるまアドベンチャー ⛄

オーロラの輝く幻想的な夜の雪世界を、雪だるまが冒険する 3D ブラウザゲームです。

![Three.js](https://img.shields.io/badge/Three.js-r160-blue)

## あそびかた

`index.html` をブラウザで開くだけで遊べます(インターネット接続が必要です。Three.js を CDN から読み込みます)。

**スマホ対応**: タッチ操作(バーチャルジョイスティック)に対応しています。GitHub Pages を有効にすると、スマホのブラウザから URL を開くだけで遊べます(`.github/workflows/pages.yml` が自動デプロイします)。

ローカルサーバーで起動する場合:

```bash
npx serve .
# または
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## おまけ: ゆきだま物理シミュレーション 🧪

`physics.html` を開くと、雪玉の物理シミュレーションで遊べます(GitHub Pages なら `<Pages の URL>/physics.html`)。

- **タップ**: 空いた場所をタップすると雪玉が生まれる
- **ドラッグ**: 雪玉をつかんで投げられる(マルチタッチ対応)
- **❄️ 10個追加**: 雪玉をまとめて追加
- **🌌 ゼロG**: 重力の ON/OFF を切り替え
- **📱 傾き**: スマホを傾けた方向に重力がかかる(iOS は許可ダイアログが出ます)
- **🔄 リセット**: 最初の状態に戻す

物理は撃力ベースの剛体球シミュレーション(重力・空気抵抗・反発係数・球同士の衝突と めり込み補正)を自前実装しています。ライブラリ不要・単一 HTML で動作します。

## ルール

- **移動**: WASD / 矢印キー(スマホは画面をドラッグ)
- **雪玉を投げる**: スペースキー(スマホは ❄️ ボタン)。自分の雪を少し使って雪玉を投げる
  - 焚き火に当てるとしばらく消えて、安全に通れる
  - 炎の精霊に当てると倒せて、**雪の結晶**を落とす(拾うと大きくなる)。精霊は時間が経つと別の場所で復活する
- **成長**: 雪の上を転がる(移動する)と少しずつ大きくなる
- **ダメージ**: 焚き火 🔥 や炎の精霊に触れると小さくなる(ノックバックあり)
- **ゲームオーバー**: 大きさがなくなる(とけてしまう)と終了
- **クリア**: フィールドに散らばる 12 個の氷のクリスタル 💎 をすべて集める
- **リトライ**: ゲームオーバー / クリア後に `R` キーまたはボタン

### ヒント

- 大きいほど動きが遅く、小さいほど速く動けます。大きさは体力でもあるので、攻めと守りのバランスを考えよう
- ダメージを受けた直後は少しの間だけ無敵になります(点滅中)
- 雪玉はサイズを消費しますが、精霊を倒して結晶を拾えばおつりが来ます。狙って投げよう

## 技術

- [Three.js](https://threejs.org/) r160(CDN・ビルド不要)
- 単一の `index.html` のみで動作
- 幻想的な夜の世界: シェーダー製オーロラ、星空(瞬きあり)、輝く月、光の精、氷晶樹、脈動する氷の花
- UnrealBloom によるポストエフェクト(発光の滲み)
- 降雪パーティクル(きらめく大粒の雪入り)、起伏のある雪原、追従カメラ、動的ライティング(焚き火・精霊・雪だるまの灯り)
